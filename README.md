## 🧠 Descripción

Invitación interactiva estilo **videojuego** basada en Minecraft.
Incluye animaciones, minijuego, efectos visuales y una mecánica de desbloqueo para revelar la invitación.

👉 No es una página común… es una **experiencia**.

---

## 🚀 Características

* 🎬 Pantalla inicial tipo **PRESS START**
* 🎥 Video de fondo inmersivo
* 🎵 Música integrada
* ⏳ Cuenta regresiva en tiempo real
* ⛏️ Minijuego de romper bloques
* 📊 Barra de progreso
* 🟩 Creeper animado (hecho en CSS)
* 💣 Explosión con sonido y efecto shake
* 🌌 Transición tipo portal
* 🏆 Invitación desbloqueable al completar el juego

## ⚙️ Cómo usar

1. Clonar o descargar el proyecto
2. Asegurarse de que todos los archivos estén en la misma carpeta
3. Abrir `index.html` en el navegador

---

## 🎮 Cómo funciona

1. Presionás **START**
2. Activás la música (opcional)
3. Jugás rompiendo bloques
4. Completás el nivel
5. 💥 Se activa el creeper
6. 🏆 Se desbloquea la invitación

---

## 🌐 Deploy (GitHub Pages)

1. Subir el repositorio a GitHub
2. Ir a **Settings → Pages**
3. Seleccionar:

   * Branch: `main`
   * Folder: `/root`
4. Guardar
